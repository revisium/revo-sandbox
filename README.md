# revo-sandbox

## Usage

```sh
node greet.js Ada    # Hello, Ada!
node greet.js        # Hello, World!
```

```sh
node farewell.js Ada    # Goodbye, Ada!
node farewell.js        # Goodbye, World!
```

```sh
node capitalize.js hello    # Hello
node capitalize.js          # (empty)
node sum.js 1 2 3   # 6
node sum.js         # 0
node reverse.js hello   # olleh
```

## Tests

```sh
node --test
```
